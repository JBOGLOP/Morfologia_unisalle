## 📚 Contenidos — Clase Puente

La clase puente **"Del Músculo al Nervio"** integra los sistemas muscular y nervioso con énfasis en aplicación clínica para optometría. Los 6 grupos expusieron:

| Grupo | Tema | Aplicación clínica |
|---|---|---|
| G1 | Unión Neuromuscular | Miastenia gravis, bloqueo neuromuscular |
| G2 | SNC vs SNP | Control pupilar, reflejo fotomotor |
| G3 | Encéfalo | ACV occipital, campimetría |
| G4 | Meninges | Herniación uncal, midriasis arreactiva |
| G5 | NC II, III, IV | Parálisis NC III, aneurisma ACom-P |
| G6 | NC V₁, VI, VII | Sensibilidad corneal, esotropía, Bell's palsy |
